jQuery — популярна JavaScript-бібліотека з відкритим сирцевим кодом(John Resig).

jQuery є найпопулярнішою бібліотекою JavaScript.

Синтаксис jQuery розроблений, щоб зробити орієнтування у навігації зручнішим завдяки вибору елементів DOM, створенню анімації, обробки подій, і розробки AJAX-застосунків.

jQuery також надає можливості для розробників, для створення плагінів у верхній частині бібліотеки JavaScript. Використовуючи ці об'єкти, розробники можуть створювати абстракції для низькорівневої взаємодії та створювати анімацію для ефектів високого рівня. Це сприяє створенню потужних і динамічних веб-сторінок.

Підключення плагіну

  1. перейти на сайт з документіцією про плагін
  2. завантажити файли і скопіювати у відповідні папки проекту
  3. скопіювати HTML розмітку
  4. підключити файли стилів у блоці HEAD
  5. підключити скрипти перед закриваючим тегом
  6. 1им повинен бути підключеним файл з бібліотекою jQuery (можна через CDN)
  7. далі скрипти скачаного плагіну
  8. далі скрипт з ініціалізацією плагіну (для того доб під’єднати функціонал скриптів до певного блоку(HTML розмітки) і він запрацював)
