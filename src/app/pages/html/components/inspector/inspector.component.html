WEB-inspector

Для розробки та налагодження локальних і віддалених веб-сторінок використовують Веб-інспектор. Він дозволяє переглядати веб-сторінки, використовуючи об'єктну модель документів, і виділяти на них елементи.

  • Щоб відкрити Веб-інспектор, виконайте наступні дії:
  • Клацніть правою кнопкою миші по елементу сторінки, потім в контекстному меню виберіть "Перевірити елемент".
  • F12

За допомогою веб інспектора ви можете здійснити покрокову перевірку коду JavaScript і швидко переглянути зміни стилів CSS.

Interface

  • кнопка вибору елемента - дає докладну інформацію про елемент.
  • кнопка вибору пристрою на якому буде відображатись сторінка
  • панель HTML в якій відображається структура сторінки
  • панель CSS зі всіма стилями вибраного вами елементу та його блочна модель. (нею зручно відслідковувати чому він відображається не так як ви це прописали)
  • консоль для відладки коду JS
