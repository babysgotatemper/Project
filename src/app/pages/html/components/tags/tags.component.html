Tag (елемент мови розмітки HTML)

Теги не чутливі до регістру BR Br br
Якщо який-небудь тег або його атрибут був написаний невірно, то браузер проігнорує подібний тег і буде відображати текст так, немов тега і не було.
Щоб розширити можливості окремих тегів і більш гнучко управляти вмістом контейнерів застосовуються атрибути тегів. a href=""

Одинарні

img src="" alt=""

br

Парні

div.../div

Блочні

характеризуються тим, що займають всю доступну ширину, висота елемента визначається його вмістом, і він завжди починається з нового рядка.

div.../div універсальний блоковий контейнер, використовується для груповання елементів/тегів і надання їм стилів і позиціонування на сторінці.


p.../p Визначає параграф (абзац) тексту


h1..6.../h1..6 Ця група тегів визначає текстові заголовки різного рівня, які показують відносну важливість секції, розташованої після заголовка.


hr Малює горизонтальну лінію

Рядкові

Рядкові елементи є безпосередньою частиною рядка.
В основному вони використовуються для зміни вигляду тексту або його виділення.
Декілька рядкових елементів на строрінці розташовуються один за одним.

a href="".../a є одним з найважливих елементів HTML і призначений для створення посилань.


img src="" alt="" призначений для відображення на веб-сторінці зображень в графічному форматі GIF, JPEG або PNG.
- src - відносний або абсолютний URI зображення (обов'язковий)
- alt - короткий опис (відображається при відключеному трафіку)


span.../h1..6 контейнер призначений для визначення рядкового елементу всередині документу. використовується для виділення окремих слів

Списки

Ordered list

встановлює нумерований список, тобто кожен елемент списку починається з числа або літери і збільшується по наростаючій.

ol
li...li
li...li
/ol
  1. list item 1
  2. list item 2
  3. list item 3

Unordered list

Встановлює маркований список, кожен елемент якого починається з невеликого символу - маркера.

ul
li...li
li...li
/ul
  • list item 1
  • list item 2
  • list item 3

li визначає окремий елемент списку. Зовнішній тег ul або ol встановлює тип списку - маркований або нумерований.
