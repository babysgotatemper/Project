Успадкування

Успадкуванням називається перенесення правил форматування для елементів, що знаходяться всередині інших.
Такі елементи є дочірніми, і вони успадковують деякі стильові властивості своїх батьків, усередині яких розташовуються.

Властивості які унаслідуються:

  • колір тексту
  • шрифт
  • міжрядкові інтервали

Правила підрахунку специфічності

Специфічність рахується по селектору.

  • кожен присутній в селекторі ідентифікатор додає до специфічності 0,1,0,0;
  • кожен клас, псевдоклас або атрибут додає до специфічності 0,0,1,0;
  • кожен елемент або псевдоелемент додає до специфічності 0,0,0,1;
p 0,0,0,1
div p 0,0,0,2
p.note 0,0,1,1
form.feedbackForm input [type = "text" ] 0,0,2,2
#conten a: hover 0,1,1,1
